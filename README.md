# community

```
使用vue全家桶开发cnode社区客户端。项目通过vue-cli搭建。 因为没有直接使用
cnode后台接口，而是自己使用koa2搭建了中间服务器，所以需要先克隆本github上的 cnode-server . 启动服务后在运行客户端
```
### 技术栈
+ Vue
+ Vue-Router
+ Vuex
+ Es6
+ Axios 跨域

### 安装依赖
```
npm install
```
### 运行启动
```
npm run serve
```
